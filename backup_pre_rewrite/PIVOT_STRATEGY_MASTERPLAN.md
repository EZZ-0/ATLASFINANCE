# EXECUTIVE SUMMARY

## The Vision

Build **one bulletproof financial analysis engine** that serves **two distinct markets**:

| | **PIVOT 1: IFA/Advisor** | **PIVOT 2: KSA First-Mover** |
|--|--------------------------|------------------------------|
| **Market** | US/Global Independent Financial Advisors | Saudi Arabia retail + mid-cap firms |
| **Users** | Small IFAs building reputation | Fresh employees 24-30 + mid-cap firms |
| **Language** | English | Arabic + English |
| **Data** | SEC + yfinance | Tadawul + Argaam |
| **Endgame** | Recurring SaaS revenue | PIF visibility + market dominance |

## Why This Works (Non-Zero-Sum)

Every feature built for one market **enhances** the other:
- Educational tooltips → Serves students (KSA) AND client education (IFA)
- White-label PDFs → Client reports (IFA) AND professional exports (KSA)
- Equation displays → Teaching tool (KSA) AND transparency for clients (IFA)
- API backend → CRM integration (IFA) AND white-label for firms (KSA)

# PIVOT 1: ATLAS FOR ADVISORS

## Target Market

**Demographics:**
- 15,000+ RIAs in the US alone
- Typically 1-5 person shops
- Managing $10M-$500M AUM
- Age: 35-55
- Pain: Can't afford Bloomberg ($24K/year) but need professional tools

**Secondary Customer: Aspiring Advisors / CFP Students**
- Age: 25-35
- Studying for CFP/CFA
- Need to understand analysis, not just pass exams
- Will become primary customers in 5-10 years

### Customer Profile


# PIVOT 2: ATLAS KSA (السوق السعودي)

## Target Market

### Primary Customers

**Segment A: Fresh Employees (24-30)**
- New graduates in finance/accounting
- Working at banks, asset managers, family offices
- Need to upskill quickly
- Willing to pay for tools that make them look competent
- Price sensitive but employer may reimburse

**Segment B: Mid-Cap Firms**
- Local investment firms
- Family offices
- Boutique asset managers
- 10-50 employees
- Need professional tools, can't afford Refinitiv/Bloomberg

**Segment C: Finance Students**
- Universities: KSU, KFUPM, Alfaisal, etc.
- Studying finance, accounting, economics
- Need practical skills beyond textbooks
- Future employees at target firms

### Ultimate Goal: PIF Visibility

The Public Investment Fund (PIF) is Saudi Arabia's $700B+ sovereign wealth fund. Getting on their radar means:
- Potential project/contract opportunities
- Validation that attracts other institutional clients
- Access to government-backed fintech initiatives

**Path to PIF:**
1. Build reputation with mid-cap firms
2. Publish notable Saudi market analysis
3. Get media coverage (Arab News, Argaam, CNBC Arabia)
4. Present at Saudi fintech events (Fintech Saudi, LEAP)
5. PIF innovation team notices → conversation starts

## Database Schema (Unified)

```sql
-- Users can be in either market
CREATE TABLE users (
    id UUID PRIMARY KEY,
    email VARCHAR(255) UNIQUE,
    name VARCHAR(255),
    market ENUM('us_ifa', 'ksa', 'both'),
    tier ENUM('free', 'student', 'pro', 'enterprise'),
    language_pref ENUM('en', 'ar', 'both'),
    created_at TIMESTAMP DEFAULT NOW()
);

-- Analyses are market-specific
CREATE TABLE analyses (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    ticker VARCHAR(20),
    market ENUM('us', 'sa'),
    data JSON,
    created_at TIMESTAMP DEFAULT NOW()
);

-- Feature flags per market
CREATE TABLE feature_flags (
    market ENUM('us_ifa', 'ksa'),
    feature VARCHAR(50),
    enabled BOOLEAN DEFAULT true
);
```
